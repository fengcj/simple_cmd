This is a html page in "public/demo/test.html"

You can change its name to "index.html" to see whether the server will return "app/views/demo/index.html.erb"


It is used for testing "public" fold in rails project, when the request come to server,
the server will first check "public" folder to see the request page can be find in it, if found,
it return this page to the client, and will not check the "routes.rb" to find contoller to handler this request.
